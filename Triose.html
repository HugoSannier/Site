Triose

Nano project // Cnam Enjmin // 4 days production

TRIOSE is a cooperative shoot 'em up for 2 players.

This project has been developpe in a week by eight peoples. It's the first student game I've worked on and it's been a real source of learning. I worked as a solo sound designer on this project and I did the SFX and the Music. I also did the integration using FMOD.

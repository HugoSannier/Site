Dans les pas de Mia

M1 project // Cnam Enjmin

Dans les pas de Mia is an enigmatic narrative phygital project playable on tablet with a children's book, in which a parent-child pair embody the sweet Mïa on her birthday.

I worked with two other sound designer on this project and I did ambiances design and integration both in FMOD and Unity using the C# API.

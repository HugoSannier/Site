Dans les pas de Mia // M1 project // Cnam Enjmin // 3 months production

Dans les pas de Mia is an enigmatic narrative phygital project playable on tablet with a children's book, in which a parent-child pair embody the sweet Mïa on her birthday.

I worked with two other sound designer on this project and I did ambiances design and integration both in FMOD and Unity using the C# API.

Puppy Guard // M1 project // Cnam Enjmin // 3 months production

Puppy Guards is a 2 - 4 players cooperative couch game in which players take on the role of museum watchmen in charge of protecting the art pieces from at best clumsy, at worst malicious visitors !

I was the only sound designer on Puppy Guard. I established the audio artistic direction with the artists and the game designers. I did the sound design, FMOD integration and Unity integration using C# API.

Puppy Guard has been select at the Busan Indie Connect in 2024.

You can find the game on itch.io :

Disco Geckos // M1 project // Cnam Enjmin // 3 months production

DISCO GECKO is a 2v2 party game where you're lost in a huge jungle nightclub and must find your friend! Use your tongue and the environment to get in the way of the opposing team... and be the first two geckos to kiss!

I was the sole sound designer working on Disco Gecko. I established the audio artistic direction in collaboration with the game artist and designer. I also managed communication with our two external composers. I used Wwise for integration, and in Unity, I worked with a dedicated tool developed by the programmers on the project.

Triose // Nano project // Cnam Enjmin // 4 days production
